1.What are the main differences between external, internal, and inline CSS?

Answer: External CSS stored in another file and referenced, internal CSS is declared in the file (typically in the head) and inline CSS Is written into elements using the “style” attribute.

2.What is the syntax for class and ID selectors?

Answer: The class selector is . and the ID selector is #.

3.How would you apply a single rule to two different selectors?

Answer: Stack them e.g.: .class1.class2{style stuff}.

4.Given an element that has an id of title and a class of primary, how would you use both attributes for a single rule?

Answer:#title.primary

5.What does the descendant combinator do?

Answer: It lets us nest selectors based on relationships.

6.Between a rule that uses one class selector and a rule that uses three type selectors, which rule has the higher specificity?

Answer:The single class selector, according to: https://www.w3schools.com/css/css_specificity.asp

7.From inside to outside, what is the order of box-model properties?

Answer: content, padding, border, margin

8.What does the box-sizing CSS property do?

Answer: It lets us include the padding and border in the total height/width of the element.

9.What is the difference between the standard and alternative box model?

Answer:In the alternative model the width is the full width of the box on the page, including padding and border, the content area is calculated as size- padding-border, while in standard the width is the width of the content area.

10.Would you use margin or padding to create more space between 2 elements?

Answer: Margin

11.Would you use margin or padding to create more space between the contents of an element and its border?

Answer: Padding

12.Would you use margin or padding if you wanted two elements to overlap each other?

Answer: Margin

13.What is the difference between a block element and an inline element?

Answer: Block elements start a new line and occupy all of it, inline does not

14.What is the difference between an inline element and an inline-block element?

Answer: Inline block has margins on all sides and can set width and height, inline cannot, it’s kind of a best of both worlds of inline and block.

15.Is an h1 block or inline?

Answer: Block

16.Is button block or inline?

Answer: Inline(though inline block on a lot of browsers)

17.Is div block or inline?

Answer: block

18.Is span block or inline?

Answer: inline

19.What’s the difference between a flex container and a flex item?

Answer: A flex item is the direct child of a flex container.

20.How do you create a flex item?

Answer: By adding children to the flex container and giving them appropriate attributes.

21.What are the 3 values defined in the shorthand flex property?

Answer: Flex-grow, Flex-shrink and Flex-basis eg: Flex: 1 0 auto

22. How do you make flex items arrange themselves vertically instead of horizontally?

Answer: Flex-direction: column (horizontal = row)

23.What is the difference between justify-content and align-items?

Answer:Align content aligns the flex lines with respect to each other, while align items aligns the flex items within the flex line, it also works without flex-wrap set to wrap and does not depend on the number of flex lines

24.How do you use flexbox to completely center a div inside a flex container?

Answer:Add: justify-content: center; align-items: center;

25.What’s the difference between justify-content: space-between and justify-content: space-around?

Answer: Space between just adds space between items, around adds space before, between and after items.
